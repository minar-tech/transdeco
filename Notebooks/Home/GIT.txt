====== GIT ======
Created Thursday 09 July 2020

**Creating a Local Git Repository**
git init
git add .
git commit -m ”Update”
git remote add origin https://github.com/transdeco-git/Transdeco
git push origin master
git remote -v
git pull origin master

**Committing and Pushing to a Remote Git Repository**
git add .					# stage
git commit -m ”Update”	# commit
git push origin mastergit	# push 

**Updating a Local Git Repository**
git pull origin master

**Updating Git**
git update-git-for-windowsgi
